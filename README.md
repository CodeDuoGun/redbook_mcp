# 二期需求拆分
目的： 发现潜在主播提高招募效率
现存问题：创作效率低、缺乏爆款标题灵感、话题使用不合理、笔记内容缺乏逻辑、优质笔记没有沉淀，内容经验无法复用。

## 需求点 构建小红书内容知识库
1、笔记抓取入库
    笔记抓取 （标题，正文，标签【话题】、作者名称、封面图、其他配图、发布时间）、笔记评分 =点赞 × 0.3+ 收藏 × 0.3+ 评论 × 0.4

    1、 笔记解析、ai生成、人工修改、发布

3、图片生成
    1、封面生成
    2、配图生成
    3、模板库



一、需求背景
目前招募侧员工需要在 小红书平台发布内容吸引潜在主播咨询，以提升主播招募效率。但在实际执行过程中存在以下问题：
1. 内容生产效率低，招募员工缺乏内容创作经验，每篇笔记需要人工寻找素材、撰写文案、配图；
2. 标题吸引力不足、笔记结构不符合小红书内容逻辑、不会合理使用话题标签；
3. 优质笔记没有沉淀，内容经验无法复用。
因此需要开发一个 内容运营后台工具，利用 AI 自动生成小红书笔记素材，帮助员工快速完成内容生产。
 二、需求目标
本项目主要实现三个目标：
1. 建立小红书笔记内容知识库
2. 通过抓取小红书平台 高互动笔记，建立结构化内容数据库。
3. 高质量笔记筛选规则：内容权重评分=点赞数×30%+收藏数×30%+评论数×40%
系统自动抓取并完成标题、封面图、笔记图片、正文文案、话题标签、作者信息、发布时间、互动数据，并自动进行 AI标签分类，形成内容知识库。 
（该知识库不直接展示给用户，作为 AI生成素材的训练数据来源。）
 三、产品定位
一个帮助运营人员快速生成高质量小红书笔记素材的 AI 内容生产工具。核心价值在于让不会写内容的人也能发爆款笔记，提升内容生产效率，形成企业级内容资产库。

1.2 笔记评分机制

1.3  AI自动识别并打标签

- 目标人群：学生、宝妈、副业人群、全职主播
六、用户功能模块
系统提供两种内容生成模式，生成内容包括：标题3-5个、 封面文案1个、正文文案1篇、标签5-8个。两种模式如下：
1. 参考笔记生成（核心功能）
用户上传小红书笔记链接，系统自动解析。解析内容包括：标题、封面、图片、文案、标签，并根据用户输入的指令输出新的笔记素材。同时，该笔记会被自动加入知识库用于AI学习。2. 通用笔记生成
2. 用户仅输入需求指令
例如：“生成一篇招募女大学生主播的笔记”或“生成一篇关于直播副业赚钱的案例分享”
 七、图片生成模块
笔记需要配图，系统提供三种方式。
1. 模板封面生成
系统提供封面模板：招募主播模板、副业赚钱模板、经验分享模板
用户只需填写：关键词，自动生成封面。
2. AI图片生成
AI根据文案自动生成：封面图、笔记配图
3. 系统提供素材库：直播间截图、主播形象、收益截图、工作环境
 八、笔记内容分类标准
1. 案例分享类。要有真人经历、收益展示和真实故事。比如“普通女生做直播3个月赚5万”
2. 干货分享类。新手小白篇“新人怎么开始做直播”、心态篇“普通人为什么适合直播副业”、直播话术篇“新人直播话术模板”、打光教程篇“直播间灯光怎么打”、粉丝维护篇“如何提高直播间复购率”。
